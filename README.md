# DescargarAplicacion

Este repositorio incluye una página simple para **GitHub Pages** que permite descargar el APK.

## Archivos

- `index.html`: página principal con botón de descarga
- `styles.css`: estilos

## Activar GitHub Pages

1. Sube este repo a GitHub.
2. Ve a **Settings** → **Pages**.
3. En **Build and deployment**, selecciona **Deploy from a branch**.
4. Elige la rama (por ejemplo `main`) y la carpeta **/(root)**.
5. Guarda. GitHub te mostrará la URL pública cuando termine de publicar.
